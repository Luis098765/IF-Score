Este é um site simples onde você poderá calcular sua média final a partir das suas notas em cada etapa.
Você não precisará de calculadora, apenas inserir suas notas referentes a cada etapa como mostrado acima.
Manual de uso:
1. Se o professor(a) atribuir apenas uma nota como média final da etapa, digite apenas ela;
Exemplo: se a sua média da etapa é apenas uma nota, a maior de duas notas ou a soma de duas ou mais notas;
2. Não use vírgulas para números quebrados, apenas pontos;
3. Note: Campo em branco = nota inexistente, campo com 0 = nota existente, ou seja 0;
Por isso, para evitar erros, exclua todos os campos em branco;
4. Para saber quanto você precisa para passar, basta adicionar notas na etapa desejada até sua média atingir 6;
